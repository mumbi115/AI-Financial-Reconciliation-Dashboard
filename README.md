# AI-Powered Financial Reconciliation & Predictive FP&A Forecasting

An end-to-end business intelligence and automated accounting solution built to streamline general ledger (GL) reconciliations, isolate transactional variances, and generate predictive budget forecasts. 

This repository contains the data pipelines, predictive modeling scripts, and Power BI dashboard layouts that transform raw ERP financial data into actionable, executive-level insights.

---

## 🚀 Project Overview

In corporate finance, manual month-end reconciliations are time-consuming, and traditional budget forecasting often ignores near-term operational momentum. This project solves both challenges:
1. **Automated Reconciliation Engine:** Automatically cross-references debit/credit legs across various accounts (Payroll, Marketing, Operations, Software) and instantly surfaces variances.
2. **Predictive FP&A Layer:** Uses a Python-driven statistical approach to forecast next-month spend by category, identifying historical budget risk indicators to support strategic CFO decision-making.

## 🛠️ Tech Stack & Methodology

* **Business Intelligence:** Power BI (DAX, Power Query, Data Modeling)
* **Predictive Modeling:** Python 3.x (`pandas`, `numpy`)
* **Data Sources:** 24 Months of historical financial transactions (Jan 2024 – Dec 2025)
* **Forecasting Method:** 3-Month Rolling Average time-series trend analysis (optimized to capture immediate operational shifts).
